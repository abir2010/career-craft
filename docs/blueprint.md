# **App Name**: CareerCraft AI

## Core Features:

- Resume Templates: Interactive resume builder with pre-designed templates.
- Email Generation: AI-powered prompt to generate tailored job application emails.
- AI Resume Review: AI resume review tool that scores the resume and provides feedback using an LLM. The LLM decides if and how it uses information about typical resume formats.
- Content Editor: User-friendly interface for editing and formatting resume content.
- Export Options: Download resume in various formats (PDF, DOCX).

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) converted to RGB hex: #3385FF for a professional and trustworthy feel.
- Background color: HSL(210, 15%, 95%) converted to RGB hex: #F0F5FF - a very light tint of blue, contributing to an airy, calm feel, suitable for a resume app.
- Accent color: HSL(180, 50%, 50%) converted to RGB hex: #40BFBF for highlighting important actions and information.
- Headline Font: 'Poppins' (sans-serif), to give a precise, contemporary look.
- Body Font: 'Inter' (sans-serif) used for the longer text, complementing 'Poppins' in the headlines, and making longer blocks of text easier to read.
- Clean and structured layout to guide the user through each step.
- Use simple, professional icons to represent different sections of the resume.
